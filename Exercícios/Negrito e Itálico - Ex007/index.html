Principais formatações

Negrito/Destaque

Nesta frase, temos um termo em negrito com a tag STRONG.

Itálico / Ênfase

Nesta frase, temos um termo em itálico com a tag I.

Texto marcado

Podemos também marcar um texto usando a tag MARK.

Podemos também, usar um estilo de marcação, com a tag MARK em SYTLE: e vai ficar assim.

Texto grande e texto pequeno

Posso usar a tag BIG (mesmo que ultrapassada) para a letra ficar grande e a tag SMALL para a letra ficar pequena.

Texto excluído

Posso criar um texto excluído usando a tag DEL para que o texto fique assim.

Texto inserido

Posso usar a tag INS para poder sublinhar a palavra que eu quiser.

Texto sobrescrito

Para criar texto que tenha algo sobrescrito, ou seja, números elevado a X, etc, é só usar a tag SUP: x20+3.

Texto subscrito

Quando quero um texto subscrito, eu uso a tag SUB para executar a ação. Por exemplo: H2O.
