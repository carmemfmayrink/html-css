Trabalhando com listas

Listas Ordenadas

Usar a tag OL, para LISTAS ORDENADAS e NUMERADAS:
  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Trocar de roupa

Posso escolher o tipo de lista "TYPE" e por onde começar "START". Exemplo: Type "x" start=4. A lista começará pelo 4. Segue exemplo:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes

Posso usar TYPE para números romanos, letras, etc

Listas não ordenadas

Usar a tag UL:
  • Arroz
  • Feijão
  • Macarrão
Existem outras formas de listas em TYPE: Circle, square. Exemplo:
  • Arroz
  • Feijão
  • Macarrão
