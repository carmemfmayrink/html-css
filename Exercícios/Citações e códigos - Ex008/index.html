Código-fonte

O comando tem que estar dentro da tag CODE para ficar assim: doc.getElementById('teste').

Se eu quiser deixar meu código organizado, com linhas em qualquer linguagem, é só usar a tag PRE.



    E soltar linha também, embora não seja o mais indicado usar a tag PRE para isso.
    (A tag indicada para espaço entre linhas é a tag BR).

Citações

Como diria o pai de um amigo: o computador é um burro muito rápido! Esta citação, pode ser colocada com " " ou com a tag Q: O computador é um burro muito rápido.

Outro tipo de citação, são aquelas usadas em livros, usando a tag BLOCKQUOTE: Segundo José, em seu livro de cores:

Se misturar as cores vermelha e amarela, a vai dar a cor laranja. E se misturar verde com amarelo, vai dar a cor verde.
